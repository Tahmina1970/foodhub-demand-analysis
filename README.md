# FoodHub Marketplace Demand Analysis

### Business Goal
Analyze restaurant and order behavior patterns to uncover demand trends and support data-driven decisions for food marketplace platforms.

### Tools & Methods
Python · Pandas · NumPy · Matplotlib · EDA · Business Insights

### Approach
- Performed exploratory data analysis
- Identified key demand drivers and customer ordering trends
- Analyzed restaurant-level performance metrics

### Result
Generated insights to support improved platform optimization and potential targeted growth strategies for restaurant onboarding and marketing.

### Next Steps / Improvement Ideas
- build Power BI dashboard version
- model demand prediction using time-series or tree models
